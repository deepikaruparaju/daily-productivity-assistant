# 🌞 Daily Productivity Assistant

Your all-in-one desktop tool to stay productive every single day — with tasks, quotes, weather updates, notifications, and basic stats!


---

## 🚀 Features

- ✅ **To-Do Task Manager** – Add, delete, toggle tasks easily.
- 🌦️ **Live Weather Updates** – Get real-time weather info by city.
- 💬 **Daily Inspirational Quotes** – Boost your mindset daily.
- 🔔 **Desktop Notifications** – Smart reminders & updates.
- 📊 **Basic Stats Tracker** – See how productive you are.
- 💡 Simple and beautiful **Tkinter GUI**
